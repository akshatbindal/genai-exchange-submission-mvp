# AI-Powered Travel Agent MVP

This project is the Minimum Viable Product (MVP) for a personalized AI travel agent, as outlined in the presentation plan. It uses a sophisticated agent-based architecture built on Google Cloud to generate hyper-personalized travel itineraries.

## Overview

- **Frontend:** A responsive web interface built with Next.js for capturing user preferences and displaying itineraries.
- **Backend:** A set of specialized AI agents built with Python, Google's Agent Development Kit (ADK), and LangGraph. The backend is designed to be deployed as containerized microservices on Google Cloud Run.
- **Data Stores:**
  - **Firestore:** Stores user profiles and session data.
  - **Neo4j:** Manages the core "taste graph" connecting users, places, and preferences.
  - **BigQuery:** Collects analytics data for future model training.
